# Weather-Dashboard# 
## Purpose

An app where you can get the weather of a specific location, and also many recipes that will allow you make a meal of your dreams while the weather is bad in your current location.


# Built With
- HTML
- CSS
- javaScript
- Bulma CSS
- Moment.js
- Weather API 
- Recipe API
